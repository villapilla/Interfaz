#Realización de la animación:

	Se utiliza una animación para desarrollar el banner,
a cada imagen se le asigna un keyframe distinto que la transforma:

*	Translación tanto en el eje X como en el Y,para las imagenes publicitarias.
*	Rotanción, para las imagenes de las flechas de movimiento.

	La animación tiene una duranción de 50s y se realiza en un ciclo infinito.
